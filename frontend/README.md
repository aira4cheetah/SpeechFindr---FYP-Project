# SpeechFindr Frontend

React + JavaScript frontend for SpeechFindr application.

## Setup

1. Create `.env` file with API URL:
```bash
VITE_API_URL=http://localhost:8000
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- Dark futuristic UI theme
- YouTube-like layout
- Real-time transcript search
- Timestamp navigation
- AI-powered summarization

## Tech Stack

- React 18
- JavaScript (ES6+)
- Vite
- TailwindCSS
- React Player

## Environment Variables

Create a `.env` file in the frontend directory with:
```
VITE_API_URL=http://localhost:8000
```

Change the URL if your backend is running on a different port or host.
